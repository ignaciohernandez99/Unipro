<h1>Instrucciones de ejecución</h1>

<ul>
  <li><b>lcs_rec.py:</b> Recursividad directa sin memorización</li>
  <li><b>lcs_divcon.py:</b> Divide y vencerás de Hirschberg</li>
  <li><b>lcs_dp.py:</b> Programación Dinámica Tabular</li>
</ul>

Asegurarse de tener instalado Python en el equipo y la librería <em>psutil</em>
```
pip install psutil
```

Tener los archivos .py y los dataset .txt en la misma carpeta.<br>
Cada dataset tiene en su nombre el número de caracteres de cada cadena de la pareja.<br>
Para ejecutar, simplemente lanzar desde la consola de comandos de Windows el script desde Python. Ejemplo:
```
python lcs_dp.py 1000.txt
```
